# TimeOut

<img width="6250" height="6250" alt="TIME OUT LOGO PNG" src="https://github.com/user-attachments/assets/ff14e9e0-df55-493f-8203-c791986b5a00" />


TimeOut, dijital refahı artırmak ve ekran bağımlılığını kontrol altına almak amacıyla geliştirilmiş yenilikçi bir mobil uygulamadır.
Uygulama, kilitlenen uygulamalara erişimi bir dil öğrenme fırsatına dönüştürerek kullanıcıların İngilizce pratiği yapmasını sağlar.

## 🚀 Temel Özellikler

* **Akıllı Uygulama Kilidi:** Kullanıcıların belirlediği uygulamalara erişimi kısıtlar ve kullanım limitleri koyar.
* **Seviyeye Göre Eğitim (A1-C2):** Uygulama kilidini açmak için kullanıcının kendi dil seviyesine uygun İngilizce kelime, çeviri veya gramer sorularını yanıtlaması gerekir.
* **Ödül Sistemi:** Sorulara verilen doğru cevaplar karşılığında belirli bir kullanım süresi (örneğin 15 dakika) kazanılır; böylece ekran süresi pasif bir tüketimden aktif bir öğrenme sürecine dönüşür.

## 🛠️ Kullanılan Teknolojiler

* **Mobil Geliştirme:** Kotlin / Java
* **Kullanıcı Arayüzü:** Jetpack Compose
* **Paket Adı:** `com.anilsevinc.timeout`
